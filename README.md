# AI 마케팅 랜딩 페이지

AI 기반 SNS 마케팅 회사를 위한 고품질의 단일페이지 홍보용 랜딩 페이지입니다.

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
- Tailwind CSS (CDN)

## 특징

- ✅ 반응형 디자인 (모바일 퍼스트)
- ✅ 접근성 최적화 (ARIA 라벨, 키보드 네비게이션)
- ✅ 스무스 스크롤 및 애니메이션
- ✅ 모바일 메뉴 지원
- ✅ 기획안에 맞춘 색상 팔레트 적용

## 색상 팔레트

- `--color-bg-dark`: #181C26 (메인 배경색)
- `--color-accent-1`: #D9D37E (주요 버튼, 핵심 강조색)
- `--color-accent-2`: #A6995B (서브 강조, 아이콘)
- `--color-neutral-1`: #BFB3A4 (보조 텍스트, 테두리)
- `--color-neutral-2`: #8C7161 (짙은 강조색, 푸터 텍스트)

## 섹션 구성

1. **Header**: 고정 네비게이션 바
2. **Hero**: 메인 헤드라인 및 CTA 버튼
3. **About**: 회사 소개
4. **Features**: 6개의 주요 서비스 카드
5. **Portfolio**: 성공 사례 그리드
6. **Contact**: 문의 폼
7. **Footer**: 저작권 정보 및 SNS 링크

## GitHub Pages 설정 방법

1. 리포지토리 페이지로 이동: https://github.com/kw900628-dot/ai-marketing-landing
2. 상단 메뉴에서 **Settings** 클릭
3. 왼쪽 사이드바에서 **Pages** 클릭
4. **Source** 섹션에서:
   - Branch를 **main**으로 선택
   - Folder를 **/ (root)**로 선택
5. **Save** 클릭
6. 몇 분 후 웹사이트가 다음 주소에서 확인 가능:
   - https://kw900628-dot.github.io/ai-marketing-landing

## 로컬 개발

로컬에서 개발하려면:

1. 리포지토리를 클론:
```bash
git clone https://github.com/kw900628-dot/ai-marketing-landing.git
```

2. 프로젝트 폴더로 이동:
```bash
cd ai-marketing-landing
```

3. 로컬 서버 실행 (예: Python):
```bash
# Python 3
python -m http.server 8000

# 또는 Node.js (http-server 설치 필요)
npx http-server
```

4. 브라우저에서 `http://localhost:8000` 접속
